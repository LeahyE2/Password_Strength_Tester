## Overview
When exploring beginner Python projects, I built a Password Strength Tester that transformed theoretical concepts into practical skills. The script prompts users to enter a password, then rigorously checks the following.
-Minimum length requirements
-Inclusion of numbers and special characters
-Capital/lowercase letter diversity

Each validation displays instant visual feedback – ✓ for passed & ✗ for failed criteria which I believe creates an intuitive user experience. This project proved perfect for honing my Python skills while avoiding complex infrastructure setups (unlike my pfSense firewall project).
Inspiration came from industry tools like Password Monster, though my solution focuses on educational transparency rather than raw cracking speed.
